## 模板模板参数

### 基本概念

模板模板参数是指模板参数本身也是一个模板。

### 示例

```cpp
template <template <typename, typename> class node, typename R>
class add {
    void print(const node<R, std::allocator<R>> &c) {
        for (auto &t : c) {
            std::cout << t << std::endl;
        }
    }
};

add<std::vector, int> ma;
```

## 模板特化

### 类型

1. **全特化** (Full Specialization)
2. **偏特化** (Partial Specialization)

**注意：** 函数只能全特化

### 编译器优先级

```
偏特化 > 全特化 > 通用模板
```

### 示例代码

```cpp
// 通用模板
template <typename A, typename B>
class text {
public:
    void print() {
        std::cout << "我一样";
    }
};

// 全特化
template <>
class text<std::string, int> {
public:
    void print() {
        std::cout << "我不一样";
    }
};

// 偏特化
template <typename A>
class text<A, int*> {
public:
    void print() {
        std::cout << "我不一样";
    }
};

// 函数特化（只能全特化）
template<typename T>
void print(const T& tri) {}

template<>
void print<int*>(int* const& value) {}
```

## 变参模板

使用参数包，通过 `...` 语法来表示。

### 基本语法

```cpp
template <typename... Args>
class as {};

template <typename T, typename... Args>
void myfun(T first, Args... args) {
    std::cout << first;
}
```

### 递归展开

```cpp
// 基础情况
void printl() {
    std::cout << "all";
}

// 递归情况
template <typename A, typename... Args>
void printl(const A &first, const Args &...args) {
    std::cout << first << " ";
    printl(args...);  // 递归调用
}

// 使用示例
// int a = 1, b = 1, c = 1, d = 1, e = 1, f = 1;
// printl(a, b, c, d, e, f);
```

## C++17 折叠表达式

### 一元折叠

对内部参数使用一个一元操作符（不常用）

```cpp
// 语法
(op ... pack)
(pack ... op)

// 示例
template <typename... Args>
bool all_true(const Args &...args) {
    return (!args && ...);  // 取非，再与；(!a) && (!b) && ...
}
```

### 二元折叠

使用二元操作符

| 类型 | 语法 | 结合方向 |
|:---|:---|:---|
| **左折叠** | `(init op ... op pack)` | 从左到右 |
| **右折叠** | `(pack op ... op init op ...)` | 从右到左 |

#### 示例：求和

```cpp
// 左折叠
template <typename... Args>
auto sum_left(const Args&... args) -> decltype((args + ...)) {
    return (args + ...);
}

// 右折叠
template <typename... Args>
auto sum_right(const Args &...args) -> decltype((... + args)) {
    return (... + args);
}
```

#### 示例：求最大值

```cpp
template <typename... Args, typename T>
auto maxn(const T &t, const Args &...args) -> decltype(std::max(t, (args, ...))) {
    return std::max(t, (args, ...));
}
```

#### 示例：打印

```cpp
template <typename... Args>
void print(const Args &...args) {
    // 方式1：逗号折叠
    ((std::cout << args), ...);

    // 方式2：使用 <<
    // (std::cout << ... << args);
}
```

## SFINAE

### 概念

**S**ubstitution **F**ailure **I**s **N**ot **A**n **E**rror（替换失败不是错误）

编译器实例化时，在替换模板参数时失败，不会将其视作编译错误，而是寻找其他可能的模板或者重载。

### 应用场景

1. **函数重载选择**
2. **类型特性检测**
3. **条件编译**

### 使用 enable_if

```cpp
#include <type_traits>

// 当 T 是整数类型时启用
template <typename T>
typename std::enable_if<std::is_integral<T>::value, void>::type
print_type(T value) {
    std::cout << "int " << std::endl;
}

// 当 T 是 const char* 时启用
template <typename T>
typename std::enable_if<std::is_same<T, const char*>::value, void>::type
print_type(T value) {
    std::cout << "const char* " << std::endl;
}

// 判断是否指针
// std::is_pointer<T>::value
```

### 判断类是否有某个成员

```cpp
template <typename T>
class has_foo {
private:
    typedef char yes[1];
    typedef char no[2];

    // U::* 是 U 的成员函数指针
    template <typename U, void (U::*)()>
    struct SFINAE {};

    template <typename U>
    static yes &test(SFINAE<U, &U::foo> *);

    template <typename U>
    static no &test(...);  // ... 指任意参数

public:
    static constexpr bool value = sizeof(test<T>(0)) == sizeof(yes);
};

// 使用
template <typename T>
typename std::enable_if<has_foo<T>::value, void>::type
call_foo(T &obj) {
    obj.foo();
    std::cout << "foo()" << std::endl;
}

class Withfoo {
public:
    void foo() { std::cout << "has\n"; }
};

class nofoo {
    // 没有 foo 成员
};
```

### 检查类型是否有 value_type

```cpp
template <typename T, typename = void>
struct has_non_void_value_type : std::false_type {};

template <typename T>
struct has_non_void_value_type<T,
    std::enable_if_t<!std::is_void_v<typename T::value_type>>
> : std::true_type {};

// 根据 has_non_void_value_type 的值进行特化
template <typename T, bool HasValueType = has_non_void_value_type<T>::value>
struct TypePrint {};

// T 有 value_type 成员
template <typename T>
struct TypePrint<T, true> {
    // 显式添加 typename 是为了告诉编译器这是类型，而不是对象
    // T::value_type -> T 的类型，例如 vector<int>::value_type = int
    static void print() {
        std::cout << "T has a member type value_type";
    }
};

// T 没有 value_type 成员
template <typename T>
struct TypePrint<T, false> {
    static void print() {
        std::cout << "no type";
    }
};

// 使用示例
struct Withint {
    using value_type = int;
};

struct noint {
    // 没有 value_type
};

// TypePrint<Withint>::print();  // 输出: T has a member type value_type
// TypePrint<noint>::print();    // 输出: no type
```

## C++20 Concept

### 概念

Concept 是 C++20 引入的约束模板参数的机制，使模板元编程更加直观和安全。

### 基本语法

```cpp
template <typename T>
concept ConceptName = 要求;
```

### 示例1：基本使用

```cpp
#include <concepts>

// 定义 concept
template <typename T>
concept Integral = std::is_integral_v<T>;

// 使用 concept 约束模板参数
template <Integral T>
void print(T a) {
    std::cout << "integral: " << a << std::endl;
}

// 或使用 requires
template <typename T>
requires Integral<T>
void print(T a) {
    std::cout << "integral: " << a << std::endl;
}
```

### 示例2：复杂约束

```cpp
// 定义可打印的概念
template <typename T>
concept printable = requires(T a) {
    { std::cout << a } -> std::same_as<std::ostream &>;
};

// 使用
template <printable T>
void print(T a) {
    std::cout << a << std::endl;
}
```

### requires 表达式

```cpp
// requires 搭配使用
// concept 限定条件，requires 排除条件

template <typename T>
concept Addable = requires(T a, T b) {
    { a + b } -> std::convertible_to<T>;
};

template <typename T>
requires Addable<T>
T add(T a, T b) {
    return a + b;
}
```

### 对比 SFINAE

| 特性 | SFINAE | Concept (C++20) |
|:---|:---|:---|
| **可读性** | 较差 | 更好 |
| **错误信息** | 难以理解 | 清晰明确 |
| **编写难度** | 较高 | 较低 |
| **编译器支持** | C++11+ | C++20+ |

## 总结

| 技术 | 引入版本 | 主要用途 |
|:---|:---:|:---|
| **模板特化** | C++98 | 为特定类型提供特殊实现 |
| **变参模板** | C++11 | 接受可变数量的模板参数 |
| **折叠表达式** | C++17 | 简化参数包展开 |
| **SFINAE** | C++11 | 类型特性检测、函数重载选择 |
| **Concept** | C++20 | 约束模板参数，提高可读性 |
