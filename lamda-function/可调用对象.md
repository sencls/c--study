## 概述

C++ 中的可调用对象包括：

- **函数指针**
- **仿函数**（函数对象）
- **Lambda 表达式**
- **std::function**
- **std::bind**

---

## 函数指针

### 基本用法

```cpp
int (*p)(int, int);

int add(int a, int b) {
    return a + b;
}

p = add;               // 等价于 p = &add;
std::cout << (*p)(1, 2);  // 等价于 p(1, 2)
```

### 优缺点

| 优点 | 缺点 |
|:---|:---|
| 简单直观 | 不能捕获上下文 |
| 适合简单的回调函数 | 指针的声明和使用较复杂 |

## 仿函数（函数对象）

仿函数是**重载了 `operator()`** 的类或结构体。

### 简单应用

```cpp
struct adder {
    int to_add;
    adder(int val) : to_add(val) {}
    int operator()(int x) {
        return x + to_add;
    }
};

adder a(2);
std::cout << a.to_add;  // 输出: 2
int b = a(1);           // 调用 operator()
std::cout << b;         // 输出: 3
```

### 优缺点

| 优点 | 说明 |
|:---|:---|
| **携带状态** | 可以拥有内部状态，通过成员变量存储数据，调用时具备上下文 |
| **灵活性高** | 可以根据需要添加更多的成员函数和变量，拓展功能 |
| **性能优化** | 编译器对仿函数优化，内联展开 |
| **类型安全** | 使用时编译期间检查 |

### 使用场景

- 需要携带状态时
- 复杂操作时（lambda 难以表达的复杂逻辑）
- 性能关键场景

### 与 STL 结合使用

```cpp
struct IsGreatThan {
    int threshold;
    IsGreatThan(int val) : threshold(val) {}
    bool operator()(int x) const {
        return x > threshold;
    }
};

IsGreatThan great(10);
std::vector<int> num = {1, 2, 10, 7};
auto it = std::find_if(num.begin(), num.end(), great);
if (it != num.end())
    std::cout << *it;
else
    std::cout << "None";
```

### 泛型比较器

```cpp
template <typename T>
struct Compare {
    bool operator()(const T& a, const T& b) const {
        return a < b;
    }
};

std::vector<int> a = {1, 1, 1, 44565, 3, 7, 8, 312};
std::sort(a.begin(), a.end(), Compare<int>());
for (const auto &t : a) {
    std::cout << t << " ";
}
```

## Lambda 表达式

### 语法

```cpp
[capture](parameters) -> return_type {
    // 函数体
}
```

| 部分 | 说明 |
|:---|:---|
| **capture** | 向外捕获需要的变量，值捕获、引用捕获、混合捕获 |
| **parameters** | 参数列表 |
| **return_type** | 可以不写，C++14 自动推导 |

### 捕获方式

| 捕获方式 | 说明 |
|:---|:---|
| `[=]` | 值捕获所有变量 |
| `[&]` | 引用捕获所有变量 |
| `[=,&val]` | 混合捕获：val 引用捕获，其他值捕获 |
| `[&,val]` | 混合捕获：val 值捕获，其他引用捕获 |
| `[=name]` | 只收集指定变量（不加等号会收集前面所有变量） |

### 示例：通过 erase 与 remove_if 实现目标删除

```cpp
int threshold = 10;
std::vector<int> a = {1, 12, 111, 4, 3, 7, 8, 312};
a.erase(std::remove_if(a.begin(), a.end(),
                       [threshold](int n) -> bool {  // true 满足条件删除
                           return n > threshold;
                       }),
        a.end());
```

### mutable 关键字

`[]() mutable` 可变 lambda 可以让值捕获的变量在函数内修改，但作用域外的值不会修改。

### 参数使用示例

```cpp
int a = 19;
int *temp = &a;
auto la = [temp](int x) -> void {
    *temp *= x;
};
la(2);
```

### 在类中使用

```cpp
class Precessor {
    int _threshold;

public:
    Precessor(int threshold) : _threshold(threshold) {}

    void process(std::vector<int> &data) {
        for (const auto &t : data)
            std::cout << t << ' ';
        std::cout << std::endl;

        data.erase(std::remove_if(data.begin(), data.end(),
                                  [this](int x) -> bool {
                                      return x <= _threshold;
                                  }),
                   data.end());

        for (const auto &t : data)
            std::cout << t << ' ';
    }
};

// 类外使用：a 是类的变量，用引用捕获
auto dfun = [&a](int x) -> void {
    a.m(23, x);
};
```

### 优缺点

| 优点 | 说明 |
|:---|:---|
| **简洁** | 代码简洁明了 |
| **灵活** | 可以捕获外部变量 |
| **性能优化** | 编译器可以优化 |
| **与 STL 无缝结合** | 可以直接用于 STL 算法 |

## std::function

`std::function`（functional 库）是 C++11 通用的可调用对象封装器，可以封装任何可调用对象，包括：

- 普通函数
- Lambda 函数
- 函数对象
- 绑定表达式

### 基本用法

```cpp
#include <functional>

// 绑定普通函数
std::function<int(int, int)> callback = add;
std::cout << callback(1, 2) << std::endl;

// 绑定仿函数
A a;
std::function<int(int, int)> fun1 = a;
std::cout << fun1(1, 2) << std::endl;

// 绑定 lambda 函数
std::function<int(int)> fun2 = [](int x) -> int { return x; };
std::cout << fun2(3) << std::endl;

std::function<void()> fun3 = []() -> void {
    std::cout << "aaaaa" << std::endl;
};
```

### 特点

| 特点 | 说明 |
|:---|:---|
| **存储各种可调用对象** | 统一的接口 |
| **支持动态改变** | 可以重新赋值 |
| **频繁调用时有开销** | 相比直接调用有额外成本 |

### 使用场景

#### 1. 回调函数

```cpp
using CallBack = std::function<void(int)>;

void triggerEvent(CallBack cb, int value) {
    cb(value);
}

// 使用 lambda
triggerEvent([](int x) -> void {
    std::cout << "aaa" << ' ' << x << std::endl;
}, 3);

// 使用仿函数
struct Printer {
    void operator()(int x) const {
        std::cout << "调用" << x << std::endl;
    }
} printt;

triggerEvent(printt, 4);
```

#### 2. 存储与调用不同类型的可调用对象

```cpp
// 使用容器类来存储
// 比如 vector<std::function<int(int,int)>>, map 之类
// emplace_back 比 push_back 少一次构造，更节省
```

## std::bind

`std::bind()` 在 functional 库中，可以绑定对象函数，同时控制一定的参数，待定参数使用 `std::placeholders` 占位。

### 基本用法

```cpp
// bind(绑定对象，参数1，参数2)
auto neew = std::bind(add, std::placeholders::_1, 2);
// 相当于 add(x, 2)
```

### 绑定类成员函数

```cpp
// 类成员函数需要取地址，类对象做第二个参数，也取地址
auto neew = std::bind(&A::m, &a, std::placeholders::_1, 2);

// 注意：静态成员函数不需要第二参数
```

### 占位符

```cpp
std::placeholders::_1  // 第一个参数
std::placeholders::_2  // 第二个参数
// ... 依此类推
```

### 完整示例

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <functional>

class A {
public:
    A() : res(0) {}
    int res;
    int m(int a, int b) {
        res = a * b;
        return res;
    }
};

int main() {
    A a;
    auto neew = std::bind(&A::m, &a, std::placeholders::_1, 2);
    auto dfun = [&a](int x) -> int {
        return a.m(23, x);
    };
    std::cout << neew(3) << ' ' << dfun(1);

    return 0;
}
```
